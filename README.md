# E-Memorandum Web App
System to manage memorandum in UTeM.

## 0. Pre-requisites
- Vue JS version 5.
- .NET Core version 6.
- Node version 18.
- NPM version 10.

## 1. Installation for Production

### 1.1. Frontend
```
$ cd web-app
$ npm run build
```

### 1.2. Backend
```
$ dotnet build
```

## 2. Running in Development

### 2.1. Frontend
```
$ cd web-app
$ npm run serve
```

### 2.2. Backend
```
$ dotnet run
```
